## 概要

HTML と CSS の基礎的なものを使ったサイト風サンプル。

一部は初心者以上の HTML・CSS を使用。

通常の`index.html` とアンケートフォームを模したフォームタグ中心の`enquete.html` を作成。

アンケートフォームの送信ボタンはダミーで実際には送信されない。

JavaScript で入力した内容をアラート表示で確認可能。
